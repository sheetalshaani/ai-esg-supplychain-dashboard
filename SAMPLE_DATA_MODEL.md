# 📁 Sample Data Model (Conceptual)

This file defines the structure of the dataset that would be used in a real implementation.

---

## 📊 Table: Supply Chain Performance

| Field Name        | Description |
|------------------|-------------|
| Region           | Geographic region of shipment |
| Supplier ID      | Unique supplier identifier |
| Product Category | Type of product shipped |
| CO2 Emissions    | Estimated emissions per shipment |
| Delivery Days    | Time taken for delivery |
| Cost             | Shipment cost |
| Delay Status     | On-time / Delayed |
