# 📁 Dataset Description

## 📊 Data Fields

| Column | Description |
|--------|------------|
| Region | Geographic location of shipment |
| Product | Product category |
| Supplier | Supplier ID |
| CO2 Emissions | Carbon emissions per shipment |
| Delivery Days | Time taken for delivery |
| Cost | Shipment cost |
| Delay Status | On-time or delayed delivery |
