# 📊 AI-Assisted ESG & Supply Chain Performance Dashboard

## 🧩 Project Overview
This project analyzes supply chain performance and sustainability (ESG) metrics using structured data and AI-assisted insights. The goal is to identify inefficiencies, track key KPIs, and support data-driven decision-making for operations and sustainability improvement.

---

## 🎯 Objective
- Analyze supply chain performance across regions and suppliers  
- Track ESG metrics such as CO2 emissions and delivery efficiency  
- Identify cost, delay, and sustainability inefficiencies  
- Use AI-assisted insights to summarize patterns and recommendations  

---

## 🛠 Tools & Technologies
- Microsoft Excel / CSV (Data preparation)  
- Power BI (Dashboard & visualization)  
- ChatGPT (AI-assisted insights & summarization)  
- GitHub (Documentation & version control)  

---

## 📊 Key Metrics Analyzed
- CO2 Emissions per shipment  
- Delivery Time (days)  
- Cost per shipment  
- Supplier performance  
- Delay rate / On-time delivery %  

---

## 📈 Key Insights
- Certain suppliers contribute disproportionately to higher emissions and delays  
- A strong correlation exists between cost inefficiency and delayed deliveries  
- Regional differences impact supply chain efficiency significantly  
- Optimizing supplier selection could reduce emissions by 15–20%  

---

## 🤖 AI Contribution
AI tools were used to:
- Identify patterns and correlations in structured data  
- Generate business insights from raw KPI data
